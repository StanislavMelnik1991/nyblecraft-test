## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

[Swagger documentation](http://localhost:5000/api/docs) available after the server starts up
## Installation

1. Install server
  
```bash
$ npm install
```

2. Create build
```bash
$ npm run build
```

3. Run Docker


## Running the app

```bash
# start server
$ npm run server:up
```

```bash
# drop server
$ npm run server:down
```
